🛰️ 우즈벡 RSS Edge Function 배포 (한 번만)

함께 받은 uz-news.ts 코드를 함수로 배포하세요. 함수 URL은 {Supabase URL}/functions/v1/uz-news로 자동 호출됩니다.

supabase functions new uz-news # 생성된 index.ts 에 uz-news.ts 내용 붙여넣기 supabase functions deploy uz-news --no-verify-jwt

--no-verify-jwt 로 배포하면 브라우저에서 인증 없이 호출됩니다 (개인용 권장). 일반 배포 시엔 anon key 가 자동 전송됩니다.

📋 Supabase 테이블 SQL (한 번만 실행)
create table if not exists news_items ( url text primary key, title text, source text, category text, desk text, country text, field text, published_at timestamptz, image text, description text, content text, lang text, summary_ko text, summary_en text, summary_uz text, tags text[], enriched boolean default false, bookmarked boolean default false, fetched_at timestamptz default now() ); -- 기존 테이블이 있으면(이전 버전 사용자): alter table news_items add column if not exists country text; alter table news_items add column if not exists field text; -- 개인용: RLS 끄기 (또는 anon 허용 정책 추가) alter table news_items disable row level security;

Supabase 대시보드 → SQL Editor 에 붙여넣고 Run. RLS를 끄는 대신 정책을 쓰려면 anon에 select/insert/update 허용 정책을 추가하세요.
